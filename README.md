## Deskripsi Aplikasi

Aplikasi ini merupakan aplikasi yang digunakan untuk mencatat transaksi keuangan. Aplikasi ini memiliki fitur untuk menambahkan transaksi, mengedit transaksi, dan menghapus transaksi.
